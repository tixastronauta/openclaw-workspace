# Learnings

Use this file for durable lessons that improve how Nyx works.

Log here only when a learning is likely to matter again.
Avoid trivial one-off notes already covered by chat history or daily memory.

## Entry template

## [LRN-YYYYMMDD-XXX] short-title

- Logged: ISO-8601 timestamp
- Status: pending | promoted | resolved | ignored
- Area: behavior | workflow | tools | security | communication | setup
- Trigger: user_feedback | mistake | recurring_pattern | research
- Summary: one-line lesson
- Action: concrete change to make
- Promote to: MEMORY.md | AGENTS.md | SOUL.md | TOOLS.md | none
- See also: optional related IDs

### Details
Concise explanation of what happened, what was learned, and why it matters.
