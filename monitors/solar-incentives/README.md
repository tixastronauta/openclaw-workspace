# Monitor: Incentivos solares Portugal

Objetivo: detetar rapidamente incentivos/candidaturas em Portugal para painéis solares, fotovoltaico, autoconsumo e eficiência energética residencial aplicável a casa privada.

Estado vivo: `state.json`.

Critérios de alerta:
- aviso oficial novo;
- abertura ou data/hora de abertura de candidaturas;
- formulário/plataforma disponível;
- detalhes de dotação, elegibilidade, voucher, comparticipação, prazos;
- notícia credível que antecipe abertura iminente.

Fontes prioritárias:
- Fundo Ambiental / Agência para o Clima;
- Portugal.gov.pt / Ministério do Ambiente e Energia;
- DGEG, ADENE, PRR/Recuperar Portugal, Portugal 2030;
- imprensa económica/energia só como sinal, nunca como confirmação única.

Notificações: alertas relevantes devem ser entregues no Telegram do Tiago (`6384494297`). Fallback: Discord #nyx-tasks apenas se a entrega por Telegram falhar.
