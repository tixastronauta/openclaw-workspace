# Porto Cultural Radar

Started: 2026-05-05

Weekly cultural radar for Tiago focused on Grande Porto events that match his interests.

Scope:
- Geography: Grande Porto
- Include: electronic music, especially drum and bass; rock; metal; relevant exhibitions
- Exclude: gastronomy, family events, theatre, cinema, cycles, classical/world music, photography-focused events
- Style: tight curation, in PT-PT, delivered via Telegram
- Timing: weekly Sunday projection for the following week, with important later events included when worth early notice
- Extra: immediate-style alerts when something is very aligned, especially DnB / strong rock-metal events
